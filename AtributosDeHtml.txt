ATRIBUTOS DE META
	<meta name="keywords" content="palabras">
		permite colocar todas las palabras claves de la pagina para que se puedad buscar mejor en el navegador
	<meta name="description" content="" >
		se usa para describir la pagina
	<meta name="author" content="">
		permite colocar el autor de la pagina web
	<meta name="copyright" content="">
		informa si la pagina tiene copyright o no, se pone el nombre de la empresa dueña de la pagina
	<meta name ="robots" content="index">
		deja que la pagina sea indexable
	<meta name ="robots" content="noindex">
		no deja que la pagina sea indexeable  o que no se muestre entre las opciones	


ATRIBUTOS DE IMAGEN
	<img src="" height""  width="" alt="" title="">
		- height"" : Alto
		- width="" : Ancho
		- alt=""   : en caso de que la imagen no se pueda mostrar aparece un titulo relacionado con la imagen	
		- title="" : sirve para darle un titulo a la imagen

		<input type ="time" value="">  :permite enviar horas
